========
mongomig
========

.. meta::
   :description: Manage migrations to MongoDB using the Relational Migrator CLI with intuitive terminal commands.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

CLI tool to manage MongoDB Relational Migrator.

The Relational Migrator CLI is a command line interface built specifically for MongoDB Atlas. You can manage your migration jobs from the terminal with short, intuitive commands.
		
Use the --help flag with any command for more info on that command.

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -h, --help
     - 
     - false
     - Shows help for mongomig.
   * - -v, --version
     - string
     - false
     - Shows the current version of mongomig.

Examples
--------

.. code-block::
   :copyable: false

   # Display the help menu for mongomig:
   mongomig --help
