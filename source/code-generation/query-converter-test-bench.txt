==========================
Query Converter Test Bench
==========================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Use Query Converter Test Bench to test converted SQL statements, 
and stored procedures. Relational Migrator uses the connections 
defined for your source and destination databases and can run the 
original and converted queries or stored procedures inside of the 
Relational Migrator application. 

Use Cases
---------

Query Converted Test Bench use cases include:

- Allowing you to compare your original and converted results 
  conveniently in a single place.
- Allowing you to quickly iterate on application modernization project 
  conversion efforts.
- Increasing your confidence in assets automatically 
  converted by the :ref:`rm-query-converter`.
