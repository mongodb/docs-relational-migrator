==========================
Query Converter Test Bench
==========================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Use Query Converter Test Bench to test SQL queries 
and stored procedures converted by :ref:`rm-query-converter`. 
Query Converter Test Bench uses the connections defined for your 
source and destination databases and allows you to run 
the original and converted queries or stored procedures.
The results from each display on the query converter 
test bench allowing for a side-by-side comparison.

Use Cases
---------

Query converted test bench allows you to:

- Compare your original and converted 
  query results in a single place.
- Increase your confidence in code automatically 
  converted by the query converter.
- See potential logic errors and edit the converted 
  code before moving it to production.

Requirements
------------

To use query converter test bench, you must 
run the Query Runner Docker container. This 
container can be run locally or hosted remotely. 
For details, see :ref:`rm-install-query-runner`.
