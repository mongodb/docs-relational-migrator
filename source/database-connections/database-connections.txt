====================
Database Connections
====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can save connection settings for relational and MongoDB databases
while creating projects and migration jobs. You can use these saved database
connections for any task that requires connecting to a relational or MongoDB
database. 

To view and manage saved connections, go to the Relational 
Migrator :guilabel:`Connections` page.
