====================
Database Connections
====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can save connection settings for both relational and MongoDB databases as
"Database Connections" on the Relational Migrator :guilabel:`Connections` page.
You can use a saved database connection for any task that requires connecting
to a relational or MongoDB database.
