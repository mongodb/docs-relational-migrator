Database Connection String
~~~~~~~~~~~~~~~~~~~~~~~~~~

Relational Migrator requires a connection string to connect to your MySQL instance.

The general form for a MySQL connection string is:

.. code-block::

   jdbc:mysql://<host:port>/<database>?<properties>

For example, the ``test`` MySQL database running on ``host1`` at port ``3306`` 
uses the following connection string:

.. code-block::

   jdbc:mysql://host1:3306/test

Alternatively, Relational Migrator can create the appropriate connection string 
based on user-provided database information.
