============================================
Save a Relational Database Connection
============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can save a new relational database connection from the Relational Migrator
:guilabel:`Connections` page, or when creating or modifying a sync job.


Steps
-----

.. procedure::
   :style:  normal

   .. step:: Add a connection
      
      Under :guilabel:`Relational` click :guilabel:`Add connection`.

   .. step:: Select the database type
      
      In the :guilabel:`Database type` drop-down, select the database.

   .. step:: To input a :ref:`connection string <rm-connection-strings>` directly:

      a. Above the :guilabel:`JDBC URI` field, enable the :guilabel:`Enter URI manually` toggle.

      #. Paste your connection string into the :guilabel:`JDBC URI` field.

   .. step:: To create a connection string by entering database information:

      Enter the following:

      .. tabs::

         .. tab:: Oracle
            :tabid: db-oracle

            .. include:: /includes/table-oracle-connection-fields.rst

         .. tab:: SQL Server
            :tabid: db-sql

            .. include:: /includes/table-sql-connection-fields.rst

         .. tab:: MySQL
            :tabid: db-mysql

            .. include:: /includes/table-mysql-connection-fields.rst
            
         .. tab:: PostgreSQL
            :tabid: db-postgresql

            .. include:: /includes/table-postgresql-connection-fields.rst

         .. tab:: Db2
            :tabid: db-db2

            .. include:: /includes/table-db2-connection-fields.rst

         .. tab:: Sybase
            :tabid: db-sybase

            .. include:: /includes/table-sybase-connection-fields.rst

   .. step:: Enter the :guilabel:`Username` and :guilabel:`Password` to use for authentication
         
      Checking :guilabel:`Save password` saves the password securely on 
      your machine. Relational Migrator automatically fills in the 
      :guilabel:`Password` field for the current project.

   .. step:: (Optional) If you are connecting to a SQL, MySQL, or PostgreSQL server, you can connect using SSL:
                        
      a. Toggle from :guilabel:`General` to :guilabel:`SSL`.
                        
      #. Select an SSL mode.

   .. step:: Specify a :guilabel:`Connection name` and optional :guilabel:`Environment tag`
         
      Environment tags color code and organize connections for 
      convenience. Leaving the drop-down empty is the same as selecting 
      the "None" tag.

   .. step:: (Optional) Click :guilabel:`Test connection`

      Relational Migrator attempts to authenticate and connect to the
      database.
         
   .. step:: Click :guilabel:`Save`
